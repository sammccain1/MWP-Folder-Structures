# Agent Lessons

Auto-updated by the agent after any user correction. Read this at the start of every session.

Each lesson follows the format:
- **What happened** — the mistake or gap
- **Root cause** — why it happened
- **Rule** — the specific behavior change going forward

---

## How to Add a Lesson

After any correction from Sam, append a new entry at the top of the `## Lessons` section using this format:

```
### YYYY-MM-DD — [Short title]
**What happened:** [What the agent did wrong or missed]
**Root cause:** [Why — missing context, wrong assumption, skipped step]
**Rule:** [Specific, actionable instruction. One sentence. Imperative mood.]
```

Lessons must be specific. "Be more careful" is not a lesson. "Always run the full test suite before marking a bug fixed, not just the failing test" is a lesson.
