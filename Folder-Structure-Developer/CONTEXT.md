## What This Template Is For

This directory is a **reusable project scaffold**. When a new developer project starts:
1. Clone this folder into the new project root
2. Fill in the CONTEXT.md files as the project takes shape
3. Use `Planning/` for specs and decisions before writing any code
4. Keep `task.md` at the root as your working checklist

**Do not build or ship code from this template directory.** It is structural scaffolding.
