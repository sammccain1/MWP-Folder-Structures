# MWP-Folder-Structures

**Model Workspace Protocol (MWP)** — a collection of opinionated, ready-to-use folder structures designed to bring clarity and consistency to AI-assisted projects.

Whether you're working as a consultant delivering data and AI solutions for clients, or a developer building passion projects, these templates give you a clean starting point that scales with your work.

---

## What's Inside

### `Folder-Structure-Consultant/`
A folder layout optimized for **AI and data consulting engagements** — client work, deliverables, research, and documentation. Designed to keep client-facing materials organized and professional.

### `Folder-Structure-Developer/`
A folder layout for **personal and passion projects** — built around a developer's workflow with room for experimentation, versioning, and iteration.

### Using Them Together
Both structures are designed to complement each other. You can adopt one or combine elements from both depending on the nature of your project.

---

## Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/sammccain1/MWP-Folder-Structures.git
   ```

2. **Copy the folder structure you need** into your project root:
   ```bash
   cp -r MWP-Folder-Structures/Folder-Structure-Consultant/ ./my-project/
   # or
   cp -r MWP-Folder-Structures/Folder-Structure-Developer/ ./my-project/
   ```

3. **Customize** the structure to fit your project's specific needs.
